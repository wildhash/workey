# Workey Demo Script

## 5-Minute Demo for Recruiters / Investors / Judges

### Opening (30s)
"I built Workey — an autonomous job-acquisition operating system with 8 specialized AI agents.
The system finds jobs, scores them against my profile, tailors my resume, drafts outreach,
and tracks applications. All without manual copy-paste."

### Show Portfolio Site (60s)
1. Open `http://localhost:3000`
2. Point to hero: "Available for AI/ML roles — this updates automatically"
3. Scroll to Projects: "These are real shipped systems — BotSpot.trade, LangGraph agents, voice AI"
4. Show Skills section: "Full-stack AI — agents, API, frontend, deployment"

### Show Command Center (90s)
1. Navigate to `/dashboard`
2. "This is the private command center — the job acquisition OS"
3. Point to Agent Fleet: "8 specialized agents, each with a clear role"
4. "Job Scout discovers listings, Scorer filters them, Tailor rewrites the resume, Cover Letter drafts 7 outreach messages"

### Run the Pipeline (90s)
1. Open terminal
2. `cd apps/worker && python main.py --task scout --mock`
3. Show output: jobs discovered → scored → classified
4. "Keyword scorer runs instantly — LLM scorer adds explainability"
5. Show score breakdown: "85+ gets auto-tailored, 70-84 goes to review queue"

### Show the API (30s)
1. Open `http://localhost:8000/docs`
2. "Full REST API — agents are callable as services"
3. "Everything is audited — every agent run logged with inputs and outputs"

### Close (30s)
"This is what I build — systems that work autonomously while I sleep.
The same architecture I apply to client AI projects.
I'm available for AI/ML engineering roles. Let's talk."

## Key Talking Points

- **Execution speed:** "I shipped this in days, not weeks"
- **Architecture:** "Modular agents, clean interfaces, production-minded design"
- **Integrity:** "No hallucinated resume content — hard rule in every prompt"
- **Breadth:** "Same skills I used for trading automation, voice AI, DeFi — platform thinking"
- **Meta point:** "I built my job search tool as a portfolio project — that's how I work"
